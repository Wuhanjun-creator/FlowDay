# FlowDay项目结构说明

## 项目概述

FlowDay是一个现代化的个人日程管理应用，旨在帮助用户以有节奏的方式规划和跟踪每日任务。该应用具有精美的UI设计，支持多语言，并提供本地数据存储功能，允许用户离线使用。

## 目录结构

```
FlowDay/
├── src/                 # 源代码目录
│   ├── auth/            # 认证相关模块
│   │   └── auth.js      # 用户认证和会话管理
│   ├── state/           # 状态管理模块
│   │   ├── store.js     # Redux风格的状态管理器
│   │   └── reducer.js   # 状态变更逻辑和初始状态定义
│   ├── storage/         # 数据存储模块
│   │   └── db.js        # IndexedDB封装和CRUD操作
│   ├── utils/           # 工具函数
│   │   └── colors.js    # 主题颜色处理
│   ├── app.js           # 主应用逻辑
│   └── i18n.js          # 国际化功能
├── app.html             # 主应用页面
├── app.js               # 主应用脚本（入口文件）
├── index.html           # 首页/营销页面
├── login.html           # 登录页面
├── login.js             # 登录逻辑
├── register.html        # 注册页面
├── register.js          # 注册逻辑
└── styles.css           # 样式文件
```

## 各模块功能详解

### 1. 用户界面模块
- **index.html**: 项目的宣传/首页，展示品牌理念并提供登录入口
- **app.html**: 主应用程序界面，包含侧边栏导航（今日概览、节奏洞察、应用设置）
- **login.html & register.html**: 用户认证相关页面，提供用户注册和登录功能

### 2. 状态管理模块 (src/state/)
- **store.js**: 实现了类似Redux的状态管理模式，提供全局状态管理功能
- **reducer.js**: 定义了应用的状态结构和状态变更逻辑，包括：
  - 任务列表管理（增删改查）
  - 用户设置（语言、主题等）
  - UI状态（过滤器、编辑状态等）
  - 默认设置配置

### 3. 数据存储模块 (src/storage/)
- **db.js**: 使用IndexedDB实现本地数据持久化
  - 提供任务的增删改查(CRUD)功能
  - 支持游客模式的数据隔离
  - 实现事务性数据库操作

### 4. 用户认证模块 (src/auth/)
- **auth.js**: 实现安全的用户认证系统
  - 使用Web Crypto API进行密码加密
  - 支持游客模式（无需注册即可使用）
  - 本地会话管理
  - 邮箱标准化处理

### 5. 国际化模块 (src/i18n.js)
- 支持中文界面
- 提供动态语言切换功能
- 统一的翻译字典管理
- 可扩展为多语言支持

### 6. 工具函数模块 (src/utils/)
- **colors.js**: 主题颜色处理和动态主题生成

### 7. 主应用逻辑 (src/app.js)
- 整合所有模块，实现完整的应用逻辑
- 处理用户交互事件
- 管理UI状态和数据流
- 与后端API的通信接口

## 技术栈与架构特点

### 技术栈
- **前端框架**: 原生JavaScript (ES6+)
- **数据持久化**: IndexedDB (客户端数据库)
- **加密技术**: Web Crypto API (用户数据安全)
- **样式**: CSS3 (包含渐变、动画等现代CSS特性)
- **设计模式**: Redux风格状态管理

### 架构特点
1. **模块化设计**: 代码按功能分层组织，便于维护和扩展
2. **离线优先**: 本地数据存储，支持离线使用场景
3. **安全性**: 采用现代加密技术保护用户数据
4. **响应式设计**: 适配不同设备和屏幕尺寸
5. **用户体验**: 流畅的动画效果和直观的操作界面
6. **国际化支持**: 内建多语言框架，方便本地化扩展

### 设计理念
- 简洁优雅的用户界面
- 专注效率的日程管理
- 本地优先的数据策略
- 安全可靠的用户认证
- 可扩展的架构设计

这个项目展示了一个功能完整、架构清晰的现代Web应用，采用了最佳实践的开发模式，各组件职责分明，易于维护和扩展。
